# Planeta GAS: Система бронирования

Бронирование тренировок и учёт продаж в Google Таблицах.

## Статус проекта
- **Sidebar**: Единый интерфейс с 3 разделами (Задачи, Продажи, Расписание).
- **Данные**: Структура 12 колонок, интеграция со `Справочником`.
- **Бизнес-логика**: Проверка занятости тренера и клиента, поиск клиентов (Typeahead), валидация времени.

## Файловая структура
- `constants.gs`: Настройки и COLS.
- `sheetDriver.gs`: Работа с Google Sheets.
- `bookingService.gs`: Логика `addTraining` и проверки.
- `logService.gs`: Логи (`CreatedRecordsLog`, `ScriptLog`).
- `ui.gs`: Меню и бэкенд для Sidebar.
- `sidebar.html`: Интерфейс с аккордеоном.

## Отключенные модули
Следующие модули вынесены в папку `gridservice/` и не используются в текущем проекте:
- `gridService.gs` — генерация сетки слотов
- `integrationService.gs` — WhatsApp интеграция

## Следующие шаги
1. **Задачи**: Доработка логики чтения задач из листа.
2. **Админ-панель**: Отмена/изменение записей через UI.
3. **Статистика**: Отчеты по посещаемости.
